# Flask Task Manager

Modern bir görev yönetim web uygulaması.

## Özellikler

- Görev ekleme
- Görev silme
- Görev tamamlama
- Öncelik sistemi (Acil / Normal / Düşük)
- Son tarih (deadline)
- Görev arama
- İstatistik ve progress bar
- Dark / Light mode
- Modern UI

## Kullanılan Teknolojiler

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript

## Kurulum

```bash
pip install flask flask-sqlalchemy

##Çalıştırma Komutu
